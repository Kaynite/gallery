# Laravel Vue Gallery

## Features

-   SPA.
-   Authentication Using Sanctum.
-   Drag and drop images (requires your account)
-   Pagination
-   Delete Images.

## Tech

-   Vue.Js 3.
-   Vuex.
-   VueRouter.
-   Laravel 8.
-   Fortify.
-   Sanctum.

## To-Do

## Installation

```sh
git clone https://github.com/Kaynite/gallery.git
cp .env.example .env
composer install
npm install
npm run dev
php artisan serve
```

For production environments...

```sh
npm run prod
```
